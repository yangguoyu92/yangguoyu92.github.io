Lowest Common Ancestor of a Binary Search Tree

Problem:

Given a binary search tree (BST), find the lowest common ancestor (LCA) of two given nodes in the BST.

According to the definition of LCA on Wikipedia: “The lowest common ancestor is defined between two nodes v and w as the lowest node in T that has both v and w as descendants (where we allow a node to be a descendant of itself).”

from LeetCode

Solution:

因为BST具有一定的性质，因此，可以通过比较节点p和节点q的值与根节点值的大小来进行求解。

  • 若节点p和节点q的值均小于根节点的值，则对左子树进行递归查找；
  • 若节点p和节点q的值均大于根节点的值，则对右子树进行递归查找；
  • 若两个节点值一个比根节点值大，一个比根节点值小，则祖先节点为根节点。

代码如下：

public class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if((root.val-p.val) * (root.val-q.val) <= 0) {
            return root;
        }
        if(p.val<root.val && q.val<root.val) {
            return lowestCommonAncestor(root.left, p, q);
        }
        if(p.val>root.val && q.val>root.val) {
            return lowestCommonAncestor(root.right, p, q);
        }
        return null;
    }
}

Tips:

  • BST：二叉查找树（Binary Search Tree），也称二叉搜索树、有序二叉树，是指一棵空树或者具有下列性质的二叉树：
    1. 若任意节点的左子树不空，则左子树上所有结点的值均小于或等于它的根结点的值；
    2. 任意节点的右子树不空，则右子树上所有结点的值均大于它的根结点的值；
    3. 任意节点的左、右子树也分别为二叉查找树。
    4. 没有键值相等的节点（英语：no duplicate nodes）。
